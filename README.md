!!간혈적인 끊김 있음 수정 예정!!

명령어
  
  !join

   자기가 들어가 있는 통화방에 참가함
  
   !play 유튜브 링크 OR 노래 제목
  
    유튜브 링크에 있는 영상 재생 / 검색한 제목의 최상단에 있는 곡 재
  
  !leave
  
    통화방 나감
  
  !pause
 
    일시 중지

  !resume
   
    다시 재생
  
  !stop
    
     영상 멈추고 플레이 리스트 초기화

  !queue

    현재 플레이 리스트 보여줌

  !loop

     반복

  !volume 1-100

    불륨 조절(현재 사용하면 봇이 망가짐)
  

  !skip

    스킵
